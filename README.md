Program Details

The MotorPH Payroll System is a console-based Java application that processes employee payroll using data from CSV files. It supports two user roles: Employee and Payroll Staff. Employees can access and view their personal information while Payroll staff are responsible for processing payroll for a single employee or all employees for a selected month. The system calculates working hours based on a standard schedule, applies a grace period and lunch deduction, then computes gross salary using hourly rates. Government deductions such as SSS, PhilHealth, Pag-IBIG, and tax are applied to determine net salary. Payroll is divided into two cut-off periods, and a monthly summary of hours worked, salary, deductions, and net pay is generated.

How System Works
The MotorPH Payrol System follows a step-by-step process to ensure accurate payroll computation.
It starts with user login, where access is given based on whether the user is an employee or part of the payroll staff.
After logging in, the system retrieves the necessary employee and attendance data from CSV files.
Payroll staff can then choose to process salaries for a specific employee or for all employees for a selected month.
The system calculates working hours based on standards schedule, attendance data is used to calculate toal working hours based on official working hours of 8:00 AM to 5:00 PM, with a 10-minute grace period for late logins, and deducts a 1-hour lunch break from daily working hours.
Once total hours are computed, the system calculates the corresponding gross salary using the employee’s hourly rate.
It then applies all required government deductions such as SSS, PhilHealth, Pag-IBIG, and withholding tax the final to determine net salary.
The computed payroll is divided into two cut-off periods within the month (1st Cut-off: 1st to 15th of the month; 2nd Cut-off: 16th to the last day of the month) for easier tracking.
All results are then compiled into a clear monthly payroll summary.
Finally, the system displays the computed details, allowing users to review salary breakdowns efficiently.
